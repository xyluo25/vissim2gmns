Convert VISSIM inpx to GMNS Format

If you want to convert VISSIM inpx to GMNS format, you can use the following code:

 1import vissim2gmns as vg
 2
 3path_inpx = "datasets/one_intersection/xl_002_001.inpx"  # Path to your inpx file
 4
 5# Prepare reference values from VISSIM,
 6# you need to update these values based on your VISSIM project
 7x_refmap = 0
 8y_refmap = 0
 9x_refnet = 0
10y_refnet = 0
11
12df_inpx = vg.vissim_inpx(path_inpx, x_refmap, y_refmap, x_refnet, y_refnet)
13
14# Save the dataframe to csv and geojson files
15df_inpx.to_file("inpx.geojson", driver="GeoJSON")
16df_inpx.to_csv("inpx.csv", index=False)
