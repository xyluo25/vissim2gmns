VISSIM2GMNS Class

VISSIM2GMNS(input_dir[, x_refmap, y_refmap, ...])

A tool to convert vissim files to geojson and csv. specifically: convert .inpx file to geopandas dataframe and a csv file convert .fzp file to geopandas dataframe and csv files convert .fhz file to csv file.
